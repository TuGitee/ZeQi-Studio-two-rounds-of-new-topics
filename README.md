# DeskTop
仿制微信发送功能和计算器的加减乘除

1.桌面部分

①把计算机和微信放到了一起，双击可以打开相应的文件

②点击其中一个可以将它移到最上方

③实现了应用的拖动

④仅可以使用删除键关闭窗口，其他按键无效

2.微信部分

①实现了按enter发送功能，ctrl+enter换行

②若为空或是只有换行会弹出警示框，并会在三秒后消失

③左边的信息也会随着输入信息一起动

④相邻的信息发送时间在2min之内不会弹出新的时间

⑤与微信聊天基本一致

3.计算机部分

①仅完成了正常的加减乘除，鲁棒性非常差，有小数点结果会保留三位小数

②左边的菜单进入退出加了动画效果

③实现了除了点击按钮，点击菜单外也可以退出菜单

④不可以连等 可以一直加减乘除 可以改变运算符号 

⑤可以清零，别的都没做...我是小废物hhh...
